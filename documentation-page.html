Variables

We will now briefly go over how variables work. In order to assign a variable, you can execute something similar to this:

x=5

What you have done is assign a value of 5 to the variable x. If you the execute

Print(x)

you will see the following output

output: 5

And just like that, you have assigned a value to your first variable. This variable can now be used in equations and can be overridden by simply executing a similar line of code but by changing 5 to another value.

Logic Operators

Time to go into Logic operators. As daunting as the name may be, note that a logic operator at its core simply connects two or more expressions such that the value of the joined expressions depends entirely on the individual expressions as well as the specific operator used. Common logic operators are the AND, OR, and NOT operators.

The AND operator will return the boolean TRUE if the two (or more) expressions that it is joining hold true. In Python, the AND operator is simply expressed as and. For example,

6>4 and 4<8
output: True

The OR operator will return a boolean TRUE if either (or any) of the expressions seperated by the operator hold true. In Python, the OR operator is represented by or. For example,

6>4 or 4==8
output: True

Lastly, the NOT operator will return true if one expression does not satisfy the other. This can get a little tricky, but its most common use is the inversion of boolean values. In Python, this operator is used as not. For example,

not True
output: False

Conditional Statements

In Python, we utilize the if, elif, and else conditional statements, typed as presented. Conditional statements will simply execute what is nested within them so long as the condition specified is met.

The if statement will execute what is specified within it if the expression that follows the statement holds true. For example.

x = 5
if x>3:
print(yes)

output: yes

The elif statement will require yet another condition and will return True if the condition specified after the if statement did not hold true but the condition specified by the elif statement does hold true. For example,

x=5
if x<3:
print(yes)
elif x>3:
print(no)

output: no

The else statement will execute what is nested within it if the if statement does not hold true and if none of the elif statements hold true. For example,

x=5
if x<5:
print(yes)
elif x>5:
print(no)
else:
print(absolutely)

output: absolutely

Loops

When coding in Python, you will make use of the For and While loops, implemented as for and while.

The for loop serves as an iterator and is useful for iterating through lists, dictionaries, or strings. For example,

x=[1,2,3]
for i in x:
print(i)

output: 1
output: 2
output: 3

The while loop, on the other hand, will execute the code nested within in so long as the condition specified holds true. For example,

x=0
while x<5:
print(x)
x += 1

output: 0
output: 1
output: 2
output: 3
output: 4
